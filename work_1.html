bwpow
(个人项目)
项目介绍
因为我之前总能忘记一些事情没做，所以我想弄一个备忘录；后来想节制一下消费，所以又搞了一个记账工具，所以这个项目就这样产生了。从设计、前端、后端、测试、发布由我一人完成。
特色功能
  1. 一键登录：登录注册一体化，利用手机号验证码即可完成
  2. 分享事情：和朋友圈一样，可设置隐私作品
  3. 记账工具：可记录每天的消费和收入
  4. 任务列表：可创建任务，自定义截止日期
  5. 主题切换：有夜间和白天模式
技术介绍
这个项目有小程序端、App端、后台管理、Node后端
App端：Hbuilder打包App，用Vue + VuetifyJS + VueRouter + Vuex + Axios + H5 Plus，H5 Plus主要是控制底层功能，比如相机，确认弹窗，选择图片
小程序端：原生小程序开发，weui样式，属于个人开发范畴
后台管理：用Vue3 + TypeScript + VuetifyJS + VueRouter + Vuex + Axios
Node后端：NestJS框架 + MongoDB
